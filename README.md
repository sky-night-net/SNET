# 🌟 SNET 3.0: High-Performance Native VPN Panel

[![Build and Release SNET 3.0](https://github.com/sky-night-net/SNET/actions/workflows/release.yml/badge.svg)](https://github.com/sky-night-net/SNET/actions/workflows/release.yml)

**SNET 3.0** — это полнофункциональная, нативная панель управления VPN, созданная для максимальной производительности и современного пользовательского опыта. Забудьте о тяжелых Docker-контейнерах: всё работает как единый бинарный файл напрямую на вашем Linux-сервере.

---

## 🔥 Ключевые возможности
- **Премиальный интерфейс**: Glassmorphism дизайн с интерактивной телеметрией в реальном времени.
- **Нативная поддержка протоколов**: AmneziaWG v1/v2, OpenVPN XOR, VLESS, VMess, Trojan, Shadowsocks.
- **Оптимизация BBR**: Автоматическая настройка TCP Congestion Control для минимальных задержек.
- **CLI Утилита**: Управление панелью через терминал одной командой `snet`.
- **Автономность**: Все зависимости встроены в один бинарный файл.

---

## 🚀 Быстрая установка (Quick Start)

Поддерживаемые ОС: **Ubuntu 22.04 / 24.04 (amd64, arm64)**.

1. **Скачайте установщик**:
   ```bash
   wget https://raw.githubusercontent.com/sky-night-net/SNET/main/install.sh
   chmod +x install.sh
   ```

2. **Запустите интерактивную установку**:
   ```bash
   sudo ./install.sh
   ```

3. **Откройте панель в браузере**:
   - Адрес: `http://ваш-ip:8080`
   - Логин: `admin`
   - Пароль: `admin` (измените после первого входа!)

---

## 🛠️ Управление через CLI (snet tool)

После установки вам доступна команда `snet` для быстрого управления:

| Команда | Описание |
| :--- | :--- |
| `snet start` | Запустить панель |
| `snet stop` | Остановить панель |
| `snet restart` | Перезагрузить панель |
| `snet status` | Проверить статус службы |
| `snet logs` | Просмотр логов в реальном времени |

---

## 🔧 Структура файлов
- `/usr/local/snet/snet` — Бинарный файл панели.
- `/etc/snet/snet.conf` — Конфигурация (Порт, IP, пути к БД).
- `/etc/snet/snet.db` — База данных пользователей и нод (SQLite).

---

## 🗑️ Удаление
Если вы захотите полностью удалить панель и все её настройки:
```bash
sudo ./install.sh uninstall
```
